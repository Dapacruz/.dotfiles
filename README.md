# .dotfiles
My Dotfiles

### Bootstrap macOS
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Dapacruz/.dotfiles/master/macos/bootstrap.sh)"
```

### Bootstrap Linux
```bash
/bin/bash -c "$(wget -O- https://raw.githubusercontent.com/Dapacruz/.dotfiles/master/linux/bootstrap.sh)"
```
